# Business metrics: growth quality over vanity

Static starter site with animated explorer (funnel, CAC/LTV/payback with LTV sensitivity bands + margin-stress LTV, animated MRR bridge + composition readout, dual cohort sparklines + forward cash snapshots) and chapter panels.

## Run locally

```bash
cd business-metrics
python3 -m http.server 3008
```

Open http://localhost:3008/
