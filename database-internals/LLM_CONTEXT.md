# Database Internals - Interactive Education

## Project Overview
Interactive educational website teaching database internals through visual, hands-on exploration.

## Topics Covered
- B-Tree indexing and operations
- Write-Ahead Logging (WAL)
- Query planning and optimization
- Transaction isolation levels
- MVCC (Multi-Version Concurrency Control)
- Join algorithms and execution
