# Distributed Systems - Interactive Education

## Project Overview
Interactive educational website teaching distributed systems concepts through visual simulations.

## Topics Covered
- Consensus algorithms (Paxos, Raft)
- CAP theorem and trade-offs
- Leader election
- Log replication
- Network partitions
- Eventual consistency

## Status

Static **site** (chapters + sidebar shell) now lives directly in this folder, alongside planning/context docs. Same shell pattern as `quant-trading`; bundled by `npm run learning-hub` from the repo root.
