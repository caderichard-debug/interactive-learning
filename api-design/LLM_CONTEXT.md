# API Design - Interactive Education

## Project Overview
Interactive educational website teaching API design concepts through visual comparisons.

## Topics Covered
- REST vs GraphQL
- API versioning strategies
- Rate limiting algorithms
- Caching strategies
- Authentication flows (OAuth, JWT)
- API documentation best practices
- Webhooks and event-driven APIs
