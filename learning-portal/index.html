Learning topics

Pick a topic below. Each lives in this repo as its own mini-site or app (quant, ML, distributed systems, API design, business metrics, networking). Design tokens and writing patterns follow the shared style and content guides.

Local: from interactive-educational-platform/, run python3 -m http.server 8000 and open /learning-portal/. All topics, one port: from the repo root, npm run learning-hub → 127.0.0.1:3000 — see docs/README.md.
