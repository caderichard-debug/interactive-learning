Takeaways
  • First meaningful byte is often dominated by DNS + TCP + TLS, not JSON parsing.
  • Loss drives retransmits and backoff—throughput collapses faster than intuition suggests on long RTT paths. Use live metrics above for the running counters.
  • H2 vs H3 · where head-of-line lives: HTTP/2 removes HTTP-layer HoL between streams, but TCP is still one byte sequence—a lost segment delays delivery of all later bytes on that connection, including other streams’ frames (TCP-shaped HoL). HTTP/3 on QUIC gives streams independent byte flows and retransmission contexts, so a stall on stream A need not block B/C the same way (you still share congestion control and link capacity).
